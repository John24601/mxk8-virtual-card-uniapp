# Paico 平台生成项目设计图 - 提示词

将下面整段内容复制到 Paico 中使用，用于生成 MXK8 虚拟卡管理小程序的页面设计图（每页一张，移动端 375px 宽）。

---

## 提示词正文（复制以下内容）

```
请为「MXK8 虚拟卡管理」小程序生成一套完整的页面设计图（线框图或高保真均可）。项目为移动端优先的虚拟卡管理应用，支持 H5/微信小程序/APP，风格简洁、偏金融/工具类。要求：每页单独一张设计图，共 11 页；画布宽度按 375px 移动端；标注主要区块与关键元素。

一、登录页
- 顶部：导航栏或 Logo，标题「登录」
- 主内容：账号输入框、密码输入框、图形验证码（验证码图片+输入框）、登录按钮
- 无底部 TabBar

二、首页
- 顶部：导航栏，标题「首页」
- 主内容从上到下：① 余额区（可用余额、卡片总余额，带金额显隐控制）② 收支统计（7天/30天/90天 Tab，收入金额、支出金额）③ 快捷入口四宫格：我的卡片、交易记录、资金流水、个人中心 ④ 最近交易列表（最多 5 条：类型、金额、时间、状态）
- 底部 TabBar：首页、卡片、我的

三、个人中心
- 顶部：导航栏，标题「我的」
- 主内容：用户信息卡片（头像、昵称、用户 ID）、菜单列表（修改密码、设置二级密码，带箭头）
- 无底部 TabBar 或与首页一致

四、修改密码
- 顶部：带返回按钮的导航栏，标题「修改密码」
- 主内容：当前密码输入框、新密码输入框、确认新密码输入框、提交按钮

五、设置二级密码
- 顶部：带返回的导航栏，标题「设置二级密码」
- 主内容：二级密码输入框、确认二级密码输入框、提交按钮

六、卡片列表
- 顶部：导航栏「我的卡片」+ 右侧「创建卡片」入口
- 主内容：卡片列表，每项包含：卡号掩码（如 ****1234）、卡片类型、可用额度/总额度、状态（正常/禁用）、操作按钮（激活/禁用、充值、查看详情）
- 可选：充值弹窗线框（快捷金额 50/100/200/500、自定义输入、确认按钮）

七、卡片详情
- 顶部：带返回的导航栏，标题「卡片详情」
- 主内容：卡片信息区（卡号掩码、类型、额度、有效期）、操作区（查看完整卡号-需二级密码、充值）、该卡交易记录列表
- 可选：查看完整卡号弹窗（二级密码输入、确认后展示完整卡号与 CVV）

八、创建卡片
- 顶部：带返回的导航栏，标题「创建卡片」
- 主内容：卡段选择、持卡人信息（姓/名、账单地址、城市/州/邮编）、联系信息（邮箱、手机、出生日期）、卡片额度与有效期、提交创建按钮

九、交易记录列表
- 顶部：导航栏，标题「交易记录」
- 主内容：筛选区（交易类型、交易状态、时间范围）、交易列表（每项：类型/名称、金额、时间、状态、商户名，可点击进详情）

十、账单详情
- 顶部：带返回的导航栏，标题「账单详情」
- 主内容：单笔交易完整信息展示（金额、类型、状态、商户、时间等）

十一、资金流水
- 顶部：导航栏，标题「资金流水」
- 主内容：流水列表（每项：类型收入/支出、金额、可用余额、状态、描述、时间）

请按以上 11 个页面分别输出设计图，保持统一的视觉规范与组件风格。
```

---

## 使用说明

1. 打开 Paico 平台，新建项目或选择「根据描述生成设计图」类能力。
2. 将上方「提示词正文」中从「请为」到「…输出设计图，保持统一的视觉规范与组件风格。」的整段复制到输入框。
3. 若平台支持「分步生成」，可拆成 11 次请求，每次只贴某一页的描述（如「一、登录页」到「无底部 TabBar」）。
4. 生成后可按页面命名保存，如：01-登录、02-首页、…、11-资金流水，便于与 [prototypes.md](../prototypes.md) 和 [PRD](../PRD.md) 对应。
